Creating and Editing New Users
##############################

This guide will help you learn how to create, delete, and edit Endless User accounts using the Endless User Settings UI.

*************
Prerequisites

To accomplish this task, you will need:

* WP Securebook
* Admin Credentials

Creating New Users
==================

To Create a New Endless User Account, follow these steps:

1. Log into the jadmin account.
2. In the Desktop searchbar at the top, type "user" and select the User Settings option

.. image:: ../_resources/CreateUser_Searchbar.jpg

3. Click the "Unlock..." Button

.. image:: ../_resources/CreateUser_UserSettings.jpg

4. Enter the jadmin password and then press the "Authenticate" button

.. image:: ../_resources/CreateUser_Authenticate.jpg

5. Click the "Add User..." button

.. image:: ../_resources/CreateUser_AddUser.jpg

6. If (and only if) you want this user account to have admin access (not advised!), click the Administrator button

.. image:: ../_resources/CreateUser_UserInfo_Admin.jpg

7. Enter the Name of the new user account

.. image:: ../_resources/CreateUser_UserInfo_Name.jpg

8. Select which Password option you want from the options listed

.. note::

    * If you want the user to set the password the next time they login, select the first option
    * If you want the user account to have no password at all, select the second option
    * If you want to manually set the account password, select the third option (recommended)


.. image:: ../_resources/CreateUser_UserInfo_SetPassword.jpg

9. If you selected option three to manually set the password, enter it now in the Password bar
10. Enter the password again in the confirm bar

.. image:: ../_resources/CreateUser_UserInfo_Password.jpg

11. Enter the password hint (blank will leave the profile with no password hint)

.. image:: ../_resources/CreateUser_UserInfo_Hint.jpg

12. If you are done entering information, click the "Add" button

.. image:: ../_resources/CreateUser_AddButton.jpg

Deleting Users
==============

To Delete an Endless User Account, follow these steps:

1. Log into the jadmin account.
2. In the Desktop searchbar at the top, type "user" and select the User Settings option

.. image:: ../_resources/CreateUser_Searchbar.jpg

3. Click the "Unlock..." Button

.. image:: ../_resources/CreateUser_UserSettings.jpg

4. Enter the jadmin password and then press the "Authenticate" button

.. image:: ../_resources/CreateUser_Authenticate.jpg

5. Click the user account you wish to remove

.. image:: ../_resources/CreateUser_RemoveUSer.jpg

6. Scroll down until you see the "Remove User..." button and click it

.. image:: ../_resources/CreateUser_RemoveButton.jpg

7. If you want to keep the user files, select the "Keep Files" button, otherwise select the "Delete Files" button

.. image:: ../_resources/CreateUser_RemoveFiles.jpg

.. note::

    Keeping the deleted users files will allow you to access them in the /home directory of the file explorer.
    
Editing User
============

To Edit the info of an Endless User Account, follow these steps:

1. Log into the jadmin account.
2. In the Desktop searchbar at the top, type "user" and select the User Settings option

.. image:: ../_resources/CreateUser_Searchbar.jpg

3. Click the "Unlock..." Button

.. image:: ../_resources/CreateUser_UserSettings.jpg

4. Enter the jadmin password and then press the "Authenticate" button

.. image:: ../_resources/CreateUser_Authenticate.jpg

5. Select the user account you wish to edit

.. image:: ../_resources/CreateUser_RemoveUser.jpg

6. To edit the profile picture, click the picture and select a new image to use

.. image:: ../_resources/CreateUser_ProfilePic.jpg

7. To edit the profile name, click the pen icon to the right of the account's current name and profile picture

.. image:: ../_resources/CreateUser_Username.jpg

8. To edit the Administrator priviledges, click the "Administrator" slide toggle (Blue means on, Grey means off)

.. image:: ../_resources/CreateUser_AdminButton.jpg

9. To edit the Parental Controls, click the "Parental Controls" button below the Administrator button

.. image:: ../_resources/CreateUser_ParentalControlsButton.jpg

.. note::

    This will require an additional entry of your admin credentials to turn on/off an account's Parental Controls
    
.. image:: ../_resources/CreateUser_ParentalControls

10. To change the Language of the profile, click the "Language" button and select a language from the list provided

.. image:: ../_resources/CreateUser_LanguageButton.jpg

11. To change the Password, click the "Password" button and enter your new password and confirm it

.. image:: ../_resources/CreateUser_EditPassword.jpg
    
12. To set your profile to automatically login, click the "Automatic Login" slide toggle

.. image:: ../_resources/CreateUser_AutomaticLogin.jpg
    
13. To view the account activity, click the "Account Activity" button

.. image:: ../_resources/CreateUser_AccountActivity.jpg
