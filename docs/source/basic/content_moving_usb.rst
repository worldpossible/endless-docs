Moving Content Using USB Device
###############################

This guide will help you learn to move content from a seperate source to a securebook using a USB device.

*************
Prerequisites

To accomplish this task, you will need:

* WP Securebook
* Black (USB) Docking Station
* USB Device
* Desired Content

Moving Content to jadmin
========================

To move content to a securebook using a USB device, follow these steps:

1. Plug the USB device into the source device from which you'll retrieve your desired content

.. image:: ../../_resources/MovingContent_USBPlugin.jpg

2. Move the desired content onto the USB device

.. image:: ../../_resources/MovingContent_ContentToUSB.jpg

3. Detach USB device from the source device

.. image:: ../../_resources/MovingContent_DetachUSBWindows.jpg

4. Power off the Securebook
5. Dock the Securebook onto the Black (USB) Docking Station

.. image:: ../../_resources/MovingContent_SecurebookDocked.jpg

6. Power on the Securebook
7. Log in as Jadmin
8. Insert the USB device to the docking station

.. image:: ../../_resources/MovingContent_USBDocked.jpg

9. Open the File Explorer Application

.. image:: ../../_resources/MovingContent_OpenFiles.png

10. Navigate to the USB device

.. image:: ../../_resources/MovingContent_NavigateUSB.jpg

11. Copy the content to the jadmin home folder

.. image:: ../../_resources/MovingContent_ContentToHome.jpg

12. Eject the USB device

.. image:: ../../_resources/MovingContent_EjectUSBeos.png

The content will now be on the securebook for jadmin to access at any time.

Moving Content to Student
=========================

To move content from a USB Device to a Student's personal Account, repeat the process shown in the "Moving Content to jadmin" tutorial but on step 7, instead of logging into jadmin, log into the desired student account, and in step 11, copy the content to the student's home folder instead of to jadmin's folder.
